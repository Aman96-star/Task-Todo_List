
            REACT TODO LIST APP


📌 Project Overview
------------------------------------------
A simple and responsive To-Do List application 
built with React to manage daily tasks.

This project demonstrates:
- React Hooks
- State management
- Event handling
- localStorage usage

------------------------------------------
🚀 Features
------------------------------------------

➕ Add new tasks
✅ Mark tasks as completed
❌ Delete tasks
📝 Edit tasks
💾 Save tasks in localStorage
🌙 Dark / Light mode (optional)
📱 Responsive design

------------------------------------------
🛠 Technologies Used
------------------------------------------

- React (Functional Components)
- useState Hook
- useEffect Hook
- CSS3
- JavaScript (ES6)
